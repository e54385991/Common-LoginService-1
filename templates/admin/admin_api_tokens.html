API接口文档
1. 更新用户余额
POST /api/external/balance
Content-Type: application/json
X-API-Key: your-token

{
  "user_id": 1,
  "amount": 100.00  // 正数增加，负数减少
}
2. 设置用户VIP等级
POST /api/external/vip-level
Content-Type: application/json
X-API-Key: your-token

{
  "user_id": 1,
  "vip_level": 3
}
3. 查询用户信息
GET /api/external/user?user_id=1
X-API-Key: your-token
